================================================================================
                               PORTAL CODER
================================================================================

BEM-VINDO AO PORTAL CODER!

O Portal Coder é uma plataforma educacional acessível desenvolvida para Coder Factory. Desenvolvido por estudantes 
da FSA com foco em acessibilidade e facilidade de uso.

================================================================================
                         COMO ABRIR E USAR O SITE
================================================================================

REQUISITOS:
- Navegador web moderno (Google Chrome)
- Conexão com internet (para recursos externos como VLibras)

INSTRUÇÕES PASSO A PASSO:

1. ABRIR O SITE:
   - Localize o arquivo "index.html" na pasta do projeto
   - Clique duas vezes no arquivo OU
   - Clique com botão direito > Abrir com > Google Chrome (Navegador recomendado para APIs de acessibilidade)

2. FAZER LOGIN:
   - Na tela inicial, você verá o formulário de login
   - Digite usuário e senha ("admin" para administrador, "teste" para aluno, ambos com a senha "1234")
   - Clique no botão "Entrar"

3. NAVEGAR NO PORTAL:
   - Após o login como aluno, você terá acesso ao menu principal
   - Após o login como administrador, você terá acesso ao dashboard administrativo e ao menu principal
   - Use os links: Sobre, Cursos, Contato
   - Acesse seu perfil clicando no ícone de usuário (👤Perfil)

================================================================================
                         RECURSOS DE ACESSIBILIDADE
================================================================================

O Portal Coder possui diversos recursos de acessibilidade:

✓ VLIBRAS: Tradução automática para Língua Brasileira de Sinais
  - Ativado automaticamente no canto inferior direito

✓ CONTROLE DE FONTE:
  - Botão "A+" para aumentar o tamanho da fonte
  - Botão "A-" para diminuir o tamanho da fonte

✓ MODO ESCURO:
  - Clique no ícone 🌙 para alternar entre modo claro e escuro

✓ NAVEGAÇÃO POR TECLADO:
  - Use TAB para navegar entre elementos
  - Use ENTER para ativar botões e links

================================================================================
                            SOLUÇÃO DE PROBLEMAS
================================================================================

PROBLEMA: O vídeo de apresentação não carrega
SOLUÇÃO: Verifique se o arquivo está em backend/public/media/apresentacao.mp4

PROBLEMA: Não consigo fazer login
SOLUÇÃO: Verifique suas credenciais com o administrador do sistema

PROBLEMA: VLibras não aparece
SOLUÇÃO: Verifique sua conexão com internet
